### ⚙️ Our Engineering Principles

We approach software development with a focus on long-term sustainability:

1.  **Readability Over Cleverness:** We write code that is easy to debug, document, and hand over to internal teams.
2.  **Performance Budgets:** We optimize assets, database queries, and API payloads to ensure applications perform well even on modest network speeds.
3.  **Security by Default:** We implement strict data validation, proper authentication handling, and secure environment configurations at every stage.
